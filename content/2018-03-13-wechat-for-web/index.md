## Introduction

仿微信网页版系统，技术栈为 MVC + JSP + Servlet，使用百度第三方编辑器框架 UEditor。

包含四个子系统：
* 网页版微信单聊子系统
* 网页版微信群聊子系统
* 网页版微信朋友圈子系统
* 网页版微信公众号子系统
